## 方案A改造要点（Task 4.2）

### 旧方案问题
- 使用 SVG foreignObject 渲染面板
- 宽度自适应内容无约束 → 太长
- 面板作为独立SVG层 → 距离主图太远
- 内容溢出 → 出现滚动

### 新方案（底部Dock）
| 改进点 | 实现 |
|--------|------|
| 渲染方式 | HTML div 替代 foreignObject |
| 容器关系 | 面板作为主图容器子元素，紧贴底部 |
| 宽度控制 | width: 100% |
| 高度控制 | max-height: 150-200px |
| 溢出处理 | overflow: hidden，内容截断 |
| 滚动控制 | overflow-y: hidden |
| 对齐方式 | text-align: left |
