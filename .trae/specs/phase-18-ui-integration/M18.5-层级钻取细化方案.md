## 1. 概述与设计原则

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| **元数据驱动** | 所有层级配置从 YAML hierarchies 配置读取，不硬编码对象类型 |
| **UI 规范** | 使用 AppButton、el-icon、语义化 CSS 变量 |
| **组件通用** | useHierarchyTypes 提供通用类型映射，可复用于任何层级 |

### 1.2 新增 composable

| Composable | 职责 |
|------------|------|
| useHierarchyTypes | 管理层级类型的配置（显示名称、层级顺序、图标等） |
| useHierarchyList | 管理钻取状态（面包屑路径、当前类型等） |

## 2. 元数据驱动设计

### 2.1 useHierarchyTypes API

```javascript
const hierarchyTypes = useHierarchyTypes()

// 从 metaObject 读取配置，或使用 DEFAULT_TYPES_CONFIG fallback
hierarchyTypes.getLabel('domain')        // '领域'
hierarchyTypes.getIcon('domain')         // 'Folder'
hierarchyTypes.getChildType('domain')   // 'sub_domain'
hierarchyTypes.hasChildren('domain')   // true
hierarchyTypes.getLevelIndex('domain') // 2
hierarchyTypes.levels.value            // [{ object_type, label, icon, children_field, is_anchor }, ...]
```

### 2.2 默认层级配置

```javascript
const DEFAULT_TYPES_CONFIG = {
  levels: [
    { object_type: 'product', label: '产品', icon: 'Box', children_field: 'versions', is_anchor: false },
    { object_type: 'version', label: '版本', icon: 'Document', children_field: 'domains', is_anchor: false },
    { object_type: 'domain', label: '领域', icon: 'Folder', children_field: 'sub_domains', is_anchor: true },
    { object_type: 'sub_domain', label: '子域', icon: 'FolderOpened', children_field: 'service_modules', is_anchor: false },
    { object_type: 'service_module', label: '服务模块', icon: 'Document', children_field: 'business_objects', is_anchor: false },
    { object_type: 'business_object', label: '业务对象', icon: 'Box', children_field: null, is_anchor: false }
  ],
  root_type: 'product',
  anchor_type: 'domain'
}
```

### 2.3 元数据驱动约束

| 约束 | 说明 |
|------|------|
| 无硬编码类型判断 | 代码中无 `if (type === 'domain')` |
| 类型映射动态读取 | `hierarchyTypes.getLabel(type)` |
| 图标映射动态读取 | `hierarchyTypes.getIcon(type)` |
| 层级顺序动态读取 | `hierarchyTypes.levels.value` |

## 5. 验收标准

### 5.1 元数据驱动验收

| 编号 | 验收项 | 检查点 |
|------|--------|--------|
| M-01 | 类型名称动态读取 | 无硬编码 `typeLabels = { domain: '领域', ... }` |
| M-02 | 类型映射通用 | 使用 `hierarchyTypes.getLabel(type)` |
| M-03 | 图标映射通用 | 使用 `hierarchyTypes.getIcon(type)` |
| M-04 | 无 if/elif 判断类型 | 代码中无 `if (type === 'domain')` |

### 5.2 UI 规范验收

| 编号 | 验收项 | 检查点 |
|------|--------|--------|
| U-01 | AppButton 复用 | 工具栏按钮使用 AppButton |
| U-02 | el-icon 复用 | 展开/收起箭头使用 el-icon |
| U-03 | el-checkbox 复用 | 复选框使用 el-checkbox |
| U-04 | CSS 变量 | 无硬编码颜色 |
