# Context 文档通用模板

> **本文件为 Context 文档通用模板,使用前请复制到对应位置并填写所有 `<占位符>`。**
> **适用范围**: `.trae/context/{utilities,services,components}/<name>.md`

## 使用方法

```bash
# 复制模板到目标位置
cp .trae/context/_TEMPLATE.md .trae/context/utilities/<name>.md

# 填写所有 <占位符>,确保 7 节全部覆盖
```

---

## 文档结构(7 节,顺序不可调整)

```
1. 职责 (What)
2. 关键函数/方法/事件 (Functions/Methods/Events)
3. 调用方 (Callers)
4. 测试覆盖现状 (Test Coverage)
5. 边界场景 (Edge Cases)
6. 易错点 (Gotchas)
7. 变更历史 (Changelog)
```

## 5. 边界场景 (Edge Cases)

列出该模块/服务/组件需要特殊处理的边界场景:

- 空输入(null / undefined / "")

- 并发同请求
- 超时
- 跨域 / 鉴权失败
