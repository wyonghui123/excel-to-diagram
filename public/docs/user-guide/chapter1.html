3 · 入门：5 步创建第一个产品版本

  1. 左侧菜单点击「产品版本管理」，进入列表页。
  2. 右上角点击「+ 新建产品」。
  3. 填写产品线名称（必填，例如 NC Cloud）和产品编码。
  4. 在产品详情页点击「+ 新建版本」，填写版本号（如 V8.0）。
  5. 选择状态为「草稿」保存，正式上线时再切到「发布」。
提示：产品编码一旦保存不可修改；版本号支持后续再编辑。

4 · 探索：关键字段速查

字段是否必填说明
产品线名称是建议与官方品牌保持一致
产品编码是全局唯一，创建后不可改
负责人是从用户下拉选择，可多人协作
版本号是支持 V*.*.* 格式，自动识别大小版本
生效日期否用于筛选"已上线"版本
备注否支持 markdown

5 · 进阶：状态流转与权限

产品版本有三种状态：

  • 草稿 (DRAFT)：仅创建者与管理员可见，可任意修改。
  • 发布 (RELEASED)：全员可见，主要字段冻结；如需修改请走变更流程。
  • 归档 (ARCHIVED)：默认隐藏，需在筛选条件主动勾选才显示。
权限提示：状态切换需要 product:release 权限，普通用户可在自己名下版本之间切换草稿/归档。
