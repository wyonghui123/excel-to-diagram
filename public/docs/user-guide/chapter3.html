3 · 入门：6 步向导

  1. 导入：上传 Excel 或选已有产品版本。
  2. 中心：勾选要在图中心显示的业务对象。
  3. 关系：选择要展示的关联类型（引用 / 依赖 / 组合）。
  4. 类型：业务对象图 / 服务模块图 二选一。
  5. 配置：布局算法（ELK / Dagre）、颜色分组、标签。
  6. 展示：生成图，导出 PNG / SVG / PDF。
提示：每步都可"上一步"返回修改；当前进度自动保存在浏览器，刷新不丢。

4 · 探索：布局算法怎么选

算法节点规模特点推荐场景
Dagre< 50速度快、树状整齐单一模块概览
ELK50 ~ 300分层清晰、避让好复杂依赖关系
D3 Force任意力导向、有机分布聚类分析

颜色分组：可按子领域、按状态、按责任人上色；同一节点最多支持 3 层分组叠加。

5 · 进阶：预设与对比

  • 保存预设：把"中心 + 关系 + 配置"打包成一个预设，下次一键复用。
  • 差异对比：同预设不同时间点跑两次，红色高亮新增节点、灰色标删除节点。
  • 批量出图：勾选 N 个产品版本，依次生成 PNG 自动打包成 ZIP。
性能提示：超过 200 个节点建议拆成多张子图分别导出，避免单图渲染卡顿。

6 · 实战：Excel 生成"供应链"主题图

Excel 模板字段：

对象编码对象名称关联对象编码关联类型
PO_001采购订单SO_001引用
PO_001采购订单WH_001依赖
SO_001销售订单WH_001依赖
WH_001入库单PO_001组合

上传后在第 2 步勾选 PO_001 作为中心，第 5 步选 ELK + 按"关联类型"上色，即可得到 4 节点 4 边的关系图。
