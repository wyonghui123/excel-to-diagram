# Excel转Diagram工具

一个从Excel文件转换为可视化业务对象关系图的工具，帮助应用架构师和需求分析人员快速理解业务对象之间的关系和模块归属。

## 功能特性

- **文件导入**：支持读取.xlsx、.xls和.csv格式的文件
- **业务对象解析**：自动解析Excel中的业务对象数据，包括对象名称、所属模块和关联关系
- **关系图生成**：基于解析的数据生成直观的业务对象关系图
- **模块信息展示**：通过颜色编码展示业务对象所属的模块
- **拖拽调整**：支持用户通过拖拽方式调整图表中对象的位置
- **图表导出**：支持将生成的图表导出为PNG图片格式
- **响应式设计**：适配不同屏幕尺寸

## 技术栈

- **前端框架**：Vue 3
- **构建工具**：Vite
- **Excel解析**：xlsx库
- **图表库**：ECharts

## 文件格式要求

### Excel文件格式

文件应包含以下列（支持中英文列名）：

- **业务对象/Object/名称**：业务对象的名称
- **模块/Module**：业务对象所属的模块
- **关联对象/RelatedObject**：与当前业务对象关联的其他对象（多个对象用分号分隔）

### CSV文件格式（重新组织后的结构）

CSV文件分为三个部分：

1. **业务对象**：包含业务对象的基本信息
2. **产品模块**：包含业务对象所属的模块信息
3. **业务对象关系**：包含业务对象之间的关系信息，包括：
   - **业务对象**：源业务对象
   - **业务对象.1**：目标业务对象
   - **关系编码**：关系的唯一标识符
   - **关系说明**：关系的详细描述

## 使用方法

1. 准备符合格式要求的文件（支持.xlsx、.xls和.csv格式）
2. 打开应用，点击"选择文件"按钮
3. 选择并上传文件
4. 系统会自动解析文件并生成业务对象关系图
5. 可以通过拖拽调整对象位置
6. 点击"导出图表"按钮将图表保存为PNG图片
7. 点击"重置布局"按钮恢复默认布局

## 常见问题

### 上传Excel文件后没有生成图表
- 检查Excel文件格式是否符合要求
- 确保Excel文件包含必要的列
- 检查浏览器控制台是否有错误信息

### 图表导出失败
- 确保浏览器支持文件下载功能
- 检查图表是否已成功生成

### 拖拽功能不工作
- 确保使用的是现代浏览器
- 检查浏览器控制台是否有错误信息
