Layer 5: 颜色计算 (GroupItem.vue / MermaidComponent.vue)
getContainerColor(container) - 分组配置页面
获取 containerData（code, elementCode, name, domain, subDomainName）
计算 centerScopeSet = new Set(centerScope)
判断 isCenter：containerCode ∈ centerScopeSet ?
判断 isCenter：container.name ∈ centerScopeMarkers.serviceModules ?
若 isCenter → 返回 centerScopeColor（如 #808080 灰色）
若非中心 → 根据 colorGroupBy 计算 colorKey
colorGroupBy='domain' → colorKey = containerData.domain
colorGroupBy='subDomain' → colorKey = containerData.subDomainName
colorGroupBy='serviceModule' → colorKey = containerData.serviceModuleName
返回 colorMapping[colorKey] 或默认颜色
颜色判断逻辑详解
// 1. 中心判断（优先级最高）
isCenter = centerScopeSet.has(containerData.code)
        || centerScopeSet.has(containerData.elementCode)
        || centerScopeSet.has(containerData.name)
        || centerScopeMarkers.serviceModules.has(containerData.name)
        || centerScopeMarkers.serviceModules.has(containerData.code)

// 2. 中心颜色
if (isCenter) return centerScopeColor  // '#808080'

// 3. 非中心颜色（按 colorGroupBy 分组）
colorKey = {
  'domain': containerData.domain,        // '供应链云'
  'subDomain': containerData.subDomainName, // '采购供应'
  'serviceModule': containerData.serviceModuleName // '条码管理'
}[colorGroupBy]

return colorMapping[colorKey]  // '#1890FF'
MermaidComponent 颜色计算
// MermaidComponent.vue 中计算节点颜色
// 使用 diagramData 中的参数：
// - centerScope: 中心范围编码
// - centerScopeColor: 中心颜色
// - colorGroupBy: 分组方式
// - colorScheme / customColors: 颜色方案

// 对每个节点：
isCenter = centerScope.includes(node.code)
color = isCenter 
  ? centerScopeColor 
  : getColorByGroup(node, colorGroupBy, colorScheme)
