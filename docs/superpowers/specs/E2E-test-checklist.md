## 测试场景1：批量删除功能

### 测试步骤

1. **进入用户管理页面**
   - [ ] 登录系统
   - [ ] 导航到系统管理 > 用户管理
   - [ ] 确认页面加载成功

2. **选择多条记录**
   - [ ] 勾选2-3条用户记录
   - [ ] 确认顶部显示"已选择 X 项"
   - [ ] 确认"删除选中"按钮可用

3. **执行批量删除**
   - [ ] 点击"删除选中"按钮
   - [ ] 确认显示确认对话框
   - [ ] 确认对话框显示正确的记录数量
   - [ ] 点击"确定"按钮

4. **验证删除结果**
   - [ ] 确认显示成功消息
   - [ ] 确认记录已被删除
   - [ ] 确认列表已刷新
   - [ ] 确认选中状态已清除

### 异常场景测试

- [ ] 未选择记录时点击"删除选中"（应显示警告）
- [ ] 删除过程中点击"取消"（应不执行删除）
- [ ] 删除不存在的记录（应显示错误消息）

## 测试场景2：批量导出功能

### 测试步骤

1. **点击导出按钮**
   - [ ] 点击"导出"按钮
   - [ ] 确认显示导出对话框

2. **配置导出选项**
   - [ ] 选择要导出的字段
   - [ ] 勾选"导出筛选后的数据"（可选）
   - [ ] 勾选"排除敏感字段"（可选）

3. **执行导出**
   - [ ] 点击"开始导出"按钮
   - [ ] 确认显示导出进度
   - [ ] 确认文件自动下载

4. **验证导出文件**
   - [ ] 打开下载的Excel文件
   - [ ] 确认包含正确的字段
   - [ ] 确认数据格式正确
   - [ ] 确认敏感字段已排除（如果勾选）

### 异常场景测试

- [ ] 未选择任何字段时导出（应显示错误）
- [ ] 导出大量数据（测试性能）
