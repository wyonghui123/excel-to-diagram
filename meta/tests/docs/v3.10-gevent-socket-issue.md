# v3.10 gevent + Python 3.14 socket 兼容性问题

## 问题概述

gevent 26.5 + Python 3.14 在 `socket.recv_into` 方法上存在已知兼容性问题，
会导致 monkey patch 之后某些 IO 操作失败。

## 影响范围

- 适用于使用 gevent 26.5 及更早版本的系统
- Python 3.14 的新 socket 实现与 gevent 的 monkey patch 存在冲突
- 主要影响 `socket.recv_into` 等底层方法

## 解决方案

1. 升级 gevent 到 26.6+ 版本（推荐）
2. 或者使用 Python 3.13 及以下版本
3. 或者禁用 gevent monkey patch

## 当前状态

v3.17+ 项目已经默认禁用 gevent 的 monkey patch（通过 `DISABLE_GEVENT_MONKEY_PATCH` 配置），
因此此问题对项目运行无影响。

## 参考资料

- gevent issue: https://github.com/gevent/gevent/issues/1871
- Python 3.14 release notes: socket 模块重构
