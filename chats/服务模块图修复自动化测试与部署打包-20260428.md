## 1. 服务模块图只显示"项目云"的问题排查

### 问题描述
用户反馈选择"管理会计"子域并全选对象后，服务模块图只显示"项目云"，其他都消失了。

### 问题根因
经过代码分析发现问题：
- **领域和子领域名称相同导致 key 冲突**：项目云（领域）= 项目云（子领域）
- 基于名称作为 key 时会导致数据覆盖

### 解决方案
1. **回退之前的修改**：之前尝试修改 `finalFilteredServiceModules` 过滤逻辑，但这不是根本解决方案
2. **添加数据校验**：在 `dataValidator.js` 中添加领域子领域名称相同校验

### 代码修改

#### dataValidator.js - 添加校验函数
```javascript
export function validateDomainSubDomainNames(rawData, previewData) {
  const items = [];
  const domainSubDomainPairs = new Map();

  // 从服务模块数据中收集领域-子领域对
  rawData.serviceComponentData?.forEach((item, index) => {
    const domain = item['领域'] || item['Domain'] || item['domain'];
    const subDomain = item['子领域'] || item['SubDomain'] || item['subDomain'];
    const smCode = item['服务模块编码'] || item['SM编码'] || item['smCode'];

    if (domain && subDomain) {
      const key = `${domain}|||${subDomain}`;
      if (!domainSubDomainPairs.has(key)) {
        domainSubDomainPairs.set(key, {
          domain,
          subDomain,
          firstRowIndex: index + 2,
          smCodes: []
        });
      }
      domainSubDomainPairs.get(key).smCodes.push(smCode);
    }
  });

  // 检查领域和子领域名称是否相同
  domainSubDomainPairs.forEach((pair, key) => {
    if (pair.domain === pair.subDomain) {
      items.push({
        level: ValidationLevel.WARNING,
        type: ValidationType.FORMAT,
        sheet: '服务模块',
        row: pair.firstRowIndex,
        field: '领域/子领域',
        value: pair.domain,
        entityCode: pair.smCodes[0],
        message: `领域"${pair.domain}"与子领域名称相同，可能导致图表渲染异常`,
        suggestion: '建议修改子领域名称，使其与领域名称不同，例如将"项目云"改为"项目管理"'
      });
    }
  });

  return items;
}
```

## 2. 架构数据管理页面状态保持问题

### 问题描述
从AA图步骤返回架构数据管理页面时，对象树和关系树的选择被清空。

### 问题根因
1. `onMounted` 会调用 `store.resetForEntry()` 重置所有选择状态
2. `handlePrevWrapper` 只在 `currentStep.value === 3` 时返回，其他步骤不会触发

### 解决方案
1. 在 `App.vue` 的 `backToArchDataManage` 中设置标志
2. 在 `AADiagramApp/index.vue` 中保存完整状态到 `sessionStorage`
3. 在 `ArchDataManageApp/index.vue` 中检测标志并恢复状态
4. 修改 `handlePrevWrapper` 条件为 `currentStep.value <= 3`

### 代码修改

#### App.vue - 设置返回标志
```javascript
backToArchDataManage() {
  console.log('[App] backToArchDataManage called')
  sessionStorage.setItem('returningFromDiagram', 'true')
  this.currentApp = 'archdatamanage'
},
```

#### AADiagramApp/index.vue - 保存完整状态
```javascript
sessionStorage.setItem('lastArchDataForDiagram', archDataStr)
```

#### ArchDataManageApp/index.vue - 恢复状态
```javascript
onMounted(async () => {
  const isReturningFromDiagram = sessionStorage.getItem('returningFromDiagram')
  console.log('[ArchDataManageApp] onMounted, isReturningFromDiagram:', isReturningFromDiagram)

  if (isReturningFromDiagram) {
    sessionStorage.removeItem('returningFromDiagram')
    store.initializing = true
    await store.fetchProducts()

    const lastArchDataStr = sessionStorage.getItem('lastArchDataForDiagram')
    if (lastArchDataStr) {
      const lastArchData = JSON.parse(lastArchDataStr)

      // 先恢复 checkedNodeIds
      if (lastArchData.checkedNodeIds?.length > 0) {
        checkedNodeIds.value = lastArchData.checkedNodeIds
      }

      // 恢复产品版本选择
      if (lastArchData.productId) {
        store.selectedProductId = lastArchData.productId
        await store.fetchVersions(lastArchData.productId)
      }
      if (lastArchData.versionId) {
        store.selectedVersionId = lastArchData.versionId
        hierarchyFilter.value = { version_id: lastArchData.versionId }
        await store.fetchFilteredTreeData(lastArchData.versionId)
      }

      // 恢复选择状态
      if (lastArchData.selectedDomains) store.selectedDomains = lastArchData.selectedDomains
      if (lastArchData.selectedDomainIds) store.selectedDomainIds = lastArchData.selectedDomainIds
      if (lastArchData.selectedSubDomains) store.selectedSubDomains = lastArchData.selectedSubDomains
      if (lastArchData.selectedSubDomainIds) store.selectedSubDomainIds = lastArchData.selectedSubDomainIds
      if (lastArchData.selectedServiceModuleIds) store.selectedServiceModuleIds = lastArchData.selectedServiceModuleIds
      if (lastArchData.selectedBusinessObjectIds) store.selectedBusinessObjectIds = lastArchData.selectedBusinessObjectIds
      if (lastArchData.relationScopeSelection) store.relationScopeSelection = lastArchData.relationScopeSelection
      if (lastArchData.activeDimension) store.activeDimension = lastArchData.activeDimension

      // 恢复 relationTypeFilter
      if (lastArchData.relationTypeFilter?.length > 0) {
        relationTypeFilter.value = lastArchData.relationTypeFilter
        activeTab.value = 'relationship'
      }
    }
    store.initializing = false
    return
  }

  store.resetForEntry()
  // ...
})
```

#### UnifiedScopePanel.vue - 添加状态恢复 watch
```javascript
const isRestoring = ref(false)

watch(() => props.initialCheckedNodeIds, (newVal) => {
  if (newVal?.length > 0) {
    console.log('[UnifiedScopePanel] Restoring checkedNodeIds from props:', newVal.length)
    isRestoring.value = true
    checkedNodeIds.value = new Set(newVal)
    nextTick(() => {
      isRestoring.value = false
    })
  }
}, { immediate: true })

watch(checkedNodeIds, (newSet) => {
  if (isRestoring.value) {
    console.log('[UnifiedScopePanel] checkedNodeIds changed during restore, skipping emit')
    return
  }
  emit('checked-change', Array.from(newSet))
}, { deep: true })
```

```javascript
const handlePrevWrapper = () => {

      emit('back-to-arch-manage')
    }
  } else {
    prevStep()
  }
}
```


```bash

```
