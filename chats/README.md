1. **在IDE中点击对应的聊天任务**
2. **输入以下指令**：
   ```

   文件格式：包含用户问题和AI回复的完整记录。
   如果文件已存在，请追加内容而不是覆盖。
   ```
3. **AI会自动填充**该 .md 文件
